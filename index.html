Pseudoclases

Estilo especiales dependiendo del contexto, posicion o estado

:root

La Pseudo-clases de CSS :root seleecciona el elemento de raiz de un documento. Aplicando a HTML, :root representa el elemento <html> y es identico al selector html, excepto que en su nivel de especificidad es mayor. Sirve para declarar variables CSS.

:hover

La Pseudo-clases de CSS :hover se representa cuando el usuario coloca el puntero sobre el elemento, pero necesariamente estaactivo. En pantallas touch :hover es problemático.

:active

La Pseudo-clases :active de CSS cubre el caso en que un elemento esta siendo activado por el usuario. cuando se interactua con el raton, se corresponde al momento en que el usuario presiona el boton del raton hasta el momento en el que lo suelta. A menudo es usado en los elemento <a> y <button> de HTML.

El orden si importa:

  1. :link {...}
    2.
  2. :visited {...}
    3.
  3. :hover {...}
    4.
  4. :active {...}
    5.

:target

La Pseudo-clases de CSS :target representa el elemento único, si existe alguno, con un id concidente con el identificador de fragmentos de la URL del documento.

Modal con CSS

:nth-child :first-child :last-child

  • nth-child: selecciona el elemento indicado que sea enésimo hijo de su elemento padre.
  • first-child: selecciona el elemento indicado que sea el primer hijo de su elemento padre.
  • last-child: selecciona el elemento indicado que sea el último hijo de su elemento padre.

:nth-of-type :first-of-type :last-of-type

  • nth-of-type: selecciona el elemento indicado que sea enésimo hijo en su tipo sin importar la jerarquia dentro del elemento padre.
  • first-of-type: selecciona el elemento indicado que sea primer hijo en su tipo sin importar la jerarquia dentro del elemento padre.
  • last-of-type: selecciona el elemento indicado que sea último hijo en su tipo sin importar la jerarquia dentro del elemento padre.

Ejemplo interactivo

:not

La Pseudo-clases de CSS de negación:not(x), es una notación funcional que toma un selector simple X como argumento. Voincide con un elemento que no esta representado. X no debe contener otro selector de negación.

:empty

La Pseudo-clases de CSS :empty corresponde a un elemento sin ningun nodo hijo. solo cuando hay nodos de elemento o texto con uno o mas caracteres dentro del elemento ese se convierte en no vacio.
